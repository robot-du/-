# 53.最大子数组和


**题目**
给你一个整数数组 nums ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。
子数组是数组中的一个连续部分。


**示例**
示例 1：
输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
输出：6
解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。

示例 2：
输入：nums = [1]
输出：1

示例 3：
输入：nums = [5,4,-1,7,8]
输出：23


**代码**
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int len = nums.size(), res = INT_MIN, cnt = 0;
        for (int i = 0; i < len; i++) {
            cnt += nums[i];      // 累加当前元素
            res = max(res, cnt); // 更新最大值
            if (cnt < 0)         // 如果当前和变为负数
                cnt = 0;         // 重置为0（放弃前面部分）
        }

        return res;
    }
};


**复杂度分析**
时间复杂度：O(n)，其中 n 为 nums 的长度。
空间复杂度：O(1)。仅用到若干额外变量。


**解释**
若 cnt ≤ 0 ，说明 cnt 对后面的结果产生 __负贡献__，即 cnt + nums[i] 还不如 nums[i] 本身大。


**知识点**
动态规划、贪心
