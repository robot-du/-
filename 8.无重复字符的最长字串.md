# 无重复字符的最长字串


**题目**
给定一个字符串 s ，请你找出其中不含有重复字符的最长 __子串__ 的长度。


**示例**
示例 1:
输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案。

示例 2:
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

示例 3:
输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
请注意，你的答案必须是子串的长度，"pwke" 是一个子序列，不是子串。


**代码**
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<int, int> m;  // 字符出现次数的哈希表
        int left = 0, n = s.size(), mx = 0;  // 左指针，字符串长度，最大长度

        for (int i = 0; i < n; i++) {
            m[s[i]]++;  // 当前字符出现次数+1
            
            // 如果当前字符重复了，移动左指针缩小窗口
            while (m[s[i]] > 1) {
                m[s[left]]--;  // 左指针字符出现次数-1
                left++;        // 左指针右移
            }
            
            // 更新最大长度
            mx = max(mx, i - left + 1);
        }
        
        return mx;
    }
};


**复杂度分析**
时间复杂度：O(N)，其中 N 是字符串的长度。左指针和右指针分别会遍历整个字符串一次。
空间复杂度：O(∣Σ∣)，其中 Σ 表示字符集（即字符串中可以出现的字符），∣Σ∣ 表示字符集的大小。


**知识点**
哈希表、双指针
