# 169.多数元素


**题目**
给定一个大小为 n 的数组 nums ，返回其中的多数元素。多数元素是指在数组中出现次数 __大于__ ⌊ n/2 ⌋ 的元素。
你可以假设数组是非空的，并且给定的数组总是存在多数元素。


**示例**
示例 1：
输入：nums = [3,2,3]
输出：3

示例 2：
输入：nums = [2,2,1,1,1,2,2]
输出：2


**代码**
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        int res = 0, cnt = 0;  // res: 候选元素, cnt: 计数器

        for (auto& x : nums) {
            if (cnt == 0) {
                res = x;       // 计数器为0时，选择当前元素作为候选
                cnt++;
            } else if (res != x)
                cnt--;         // 当前元素与候选不同，抵消一个计数
            else
                cnt++;         // 当前元素与候选相同，增加计数
        }

        return res;            // 返回候选元素（多数元素）
    }
};


**复杂度分析**
时间复杂度：O(n)。Boyer-Moore 算法只对数组进行了一次遍历。
空间复杂度：O(1)。Boyer-Moore 算法只需要常数级别的额外空间。


**解释**
Boyer-Moore 算法：
抵消策略： __不同元素__ 相互抵消
多数必胜：由于多数元素数量 __> n/2__ ，最终一定幸存
幸存者原理：最后剩下的候选就是多数元素


**知识点**
Boyer-Moore 算法
