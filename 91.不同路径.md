# 61.不同路径


**题目**
一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为 “Start” ）。
机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。
问总共有多少条不同的路径？


**示例**
示例 1：
![alt text](image-91.png)
输入：m = 3, n = 7
输出：28

示例 2：
输入：m = 3, n = 2
输出：3
解释：
从左上角开始，总共有 3 条路径可以到达右下角。
1. 向右 -> 向下 -> 向下
2. 向下 -> 向下 -> 向右
3. 向下 -> 向右 -> 向下

示例 3：
输入：m = 7, n = 3
输出：28

示例 4：
输入：m = 3, n = 3
输出：6

## 方法一（动态规划）
**代码**
class Solution {
public:
    int uniquePaths(int m, int n) {
        int rows = m, cols = n;

        // 创建DP表，res[i][j]表示从(0,0)到(i,j)的路径数
        vector<vector<int>> res(rows, vector<int>(cols, 0));
        
        // 初始化第一列：从起点到第一列任何位置只有1条路径（一直向下）
        for (int i = 0; i < rows; i++) {
            res[i][0] = 1;
        }
        
        // 初始化第一行：从起点到第一行任何位置只有1条路径（一直向右）
        for (int i = 0; i < cols; i++) {
            res[0][i] = 1;
        }
        
        // 填充DP表的其余部分
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                // 到达(i,j)的路径数 = 从上面来的路径数 + 从左边来的路径数
                res[i][j] = res[i - 1][j] + res[i][j - 1];
            }
        }

        // 返回终点(m-1, n-1)的路径数
        return res[rows - 1][cols - 1];
    }
};


**复杂度分析**
时间复杂度：O(mn)，其中 m 为数组行数， n 为数组列数，需要遍历一次数组。
空间复杂度：O(mn)，可以优化
